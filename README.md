```bash
npm run dev

yarn dev

pnpm dev
# or
bun dev
```

## Prisma 
Create db with prisma
```bash
npx prisma init   
```

Pull the schema from mongo db atlas
```bash
npx prisma db pull   
```

Push the schema from local to mongo db atlas
```bash
npx prisma db push   
```

Create prisma client
```bash
npx prisma generate
```
